A web app to view and filter microbial species. Built with Next.js, TypeScript, and Tailwind CSS.

**Live Demo**: [https://hello-biome-test-task-pi.vercel.app/](https://hello-biome-test-task-pi.vercel.app/)

## ✨ Features

- **Infinite Scroll**: Loads data as you scroll down.
- **Filtering**: Filter by microbe type (Bacteria, Fungi).
- **Sorting**: Sort by name or abundance.
- **Mobile Friendly**: Works on all screen sizes.

## 🚀 Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```
2.  **Run the app**:
    ```bash
    npm run dev
    ```
